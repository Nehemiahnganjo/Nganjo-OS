# Changelog

## v1.0 Lite — "Arise" (2026-04-07)

first release. its a lite version, not trying to be everything.

### whats in it

- arch base with linux kernel
- gnome 49 on wayland
- gdm with auto login in live session
- pipewire audio
- bluetooth
- networkmanager
- zram swap
- cpu performance tuning
- io scheduler tuning
- calamares installer (built from aur during iso build)
- custom plymouth theme
- custom wallpaper + gdm background
- papirus dark icons
- bibata modern ice cursor
- adw-gtk3 theme
- dash to dock, appindicator, blur my shell extensions
- firefox preinstalled
- brave optional via aur
- flatpak + flathub ready
- fastfetch, btop, htop, bat, eza, fzf, ripgrep, fd, tmux
- ufw firewall
- apparmor
